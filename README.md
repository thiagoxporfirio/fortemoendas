# Forte Moendas — Site (scaffold)

Projeto inicial em React + TypeScript (Vite) para o site institucional da Forte Moendas.

Como rodar localmente:

1. Instalar dependências

```bash
npm install
```

2. Rodar em modo de desenvolvimento

```bash
npm run dev
```

3. Build de produção

```bash
npm run build
```

Arquivos importantes:

- [package.json](package.json)
- [vite.config.ts](vite.config.ts)
- [src/App.tsx](src/App.tsx)
